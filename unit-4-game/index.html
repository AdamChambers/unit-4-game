Crystals Collector

You will be given a random number at the start of the game


There are four crystals below. By clicking on a crystal you will be adding a specific amount of points to your score


You win the game by collecting the same amount of points (crystals) as the randomly generated number. You lose if your total crystals collected goes above the computers randomly generated number


The value of each crystal is hidden until you click it


Each time the game starts, the game will change the value of each crystal
